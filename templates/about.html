關於植物辨識系統

專案介紹

植物辨識系統是一個使用深度學習技術來辨識12種不同植物的應用程式。本系統使用TensorFlow框架訓練的卷積神經網路模型，能夠從照片中識別植物種類，並提供相關資訊。

技術架構

  • 後端框架： Python Flask
  • 深度學習框架： TensorFlow
  • 圖像處理： OpenCV, Pillow
  • 前端技術： HTML, CSS, JavaScript

功能特色

  • 支援12種植物的辨識
  • 支援上傳照片或使用相機拍攝
  • 提供植物詳細資訊和照顧提示
  • 保存辨識歷史記錄
  • 簡約直觀的使用者介面
  • 支援多用戶同時使用

使用方法

  1. 在首頁選擇上傳照片或使用相機拍攝
  2. 系統會自動分析照片並辨識植物種類
  3. 查看辨識結果和植物詳細資訊
  4. 可以在歷史記錄頁面查看過去的辨識結果
  5. 在植物資訊頁面瀏覽所有支援的植物資訊
